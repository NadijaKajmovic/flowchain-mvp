Build a single-page web application called FlowChain.
FlowChain helps importing SMEs reduce settlement friction and understand working-capital needs.
The application must consist of a single HTML file containing HTML, CSS and JavaScript.

Features:
1. User enters supplier invoice amount, transport cost, import duty, VAT percentage, expected sales revenue, current cash balance and sales period.

2. App generates a settlement plan showing:
- Supplier payment
- Transport payment
- Import duty
- VAT reserve
- Total obligations

3. App calculates the projected cash gap.

4. App recommends:
- Financing amount
- Estimated monthly repayment

5. App shows which external integrations are simulated:
- Open Banking API
- Supplier API
- Transport API
- Tax/Customs API
- FX Provider

No backend.
No database.
No real APIs.
No external libraries.

The UI should look clean, investor-oriented and easy to demo.
