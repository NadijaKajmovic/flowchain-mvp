# FlowChain MVP

FinTech: Business Models and Applications (BM26BAM) – Assignment 2

## Overview

FlowChain is a single-page MVP that helps importing SMEs reduce settlement friction and understand working-capital needs.

The application transforms purchase order information into:

- A settlement plan
- A VAT reserve calculation
- A projected cash-gap analysis
- A financing recommendation
- An overview of simulated external integrations

## Features

- Supplier invoice input
- Transport cost input
- Import duty input
- VAT percentage input
- Expected sales revenue input
- Current cash balance input
- Sales period input
- Settlement plan generation
- Cash-gap calculation
- Financing recommendation
- Simulated Open Banking, supplier, transport, customs and FX integrations

## Technical Details

- Single HTML file
- Inline CSS
- Inline JavaScript
- No backend
- No database
- No external libraries

## Repository Structure

- `flowchain.html` – MVP application
- `prompt.md` – Initial prompt used to generate the MVP
- `CLAUDE.md` – Instructions for the AI assistant
- `chat_log.md` – Collaboration history with the AI assistant
- `README.md` – Project documentation

## Disclaimer

This MVP demonstrates the FlowChain concept and does not execute real payments, customs filings, foreign exchange transactions or financing approvals. External integrations are simulated and would require real partners, APIs and regulatory licenses in a production environment.
